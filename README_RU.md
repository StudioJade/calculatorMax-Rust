Мощное приложение-калькулятор, написанное на Rust, с графическим интерфейсом и расширенными математическими возможностями.

## Особенности

- **Безопасная оценка выражений**: Использует крейт `meval` для безопасного анализа математических выражений
- **Расширенные математические функции**: Поддерживает тригонометрические, логарифмические и другие продвинутые функции
- **Геометрические вычисления**: Расчет площади треугольников, прямоугольников, кругов и трапеций
- **Специальные формулы**: Формула Герона для площади треугольника, теорема Пифагора
- **Отслеживание истории**: Сохраняет историю предыдущих вычислений
- **Настраиваемая безопасность**: Переключение между режимами безопасной и расширенной оценки
- **Современный графический интерфейс**: Создан с помощью egui для отзывчивого пользовательского интерфейса

## Установка

### Предварительные требования

- Инструментальная цепочка Rust (рекомендуется последняя стабильная версия)

### Сборка

```bash
cargo build --release
```

### Запуск

```bash
cargo run
```

## Использование

1. Введите математическое выражение в поле ввода
2. Нажмите "Вычислить" или клавишу Enter
3. Просмотрите результат в поле результата
4. Используйте кнопку "История" для просмотра предыдущих вычислений
5. Переключите "Безопасный режим" для управления ограничениями оценки

## Математические функции

Калькулятор поддерживает широкий спектр функций:

- Базовая арифметика: `+`, `-`, `*`, `/`, `%`, `^`
- Константы: `pi`, `e`
- Тригонометрические: `sin`, `cos`, `tan` и т.д.
- Логарифмические: `log`, `log10`, `ln`

## Архитектура

Приложение структурировано на несколько модулей:

- `calculator`: Основная логика вычислений
- `ui`: Графический интерфейс пользователя
- `config`: Настройки приложения
