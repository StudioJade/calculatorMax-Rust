# 計算機 Max - Rust 版本

一個功能強大的計算器應用程式，使用 Rust 編寫，具有圖形用戶界面和廣泛的數學功能。

## 功能特色

- **安全表達式評估**：使用 `meval` crate 進行安全的數學表達式解析
- **豐富數學函數**：支援三角函數、對數函數和其他高級函數
- **幾何計算**：三角形、矩形、圓形和梯形的面積計算
- **特殊公式**：海倫公式計算三角形面積、畢氏定理
- **歷史記錄追踪**：保存先前的計算記錄
- **可配置安全性**：可在安全模式和擴展模式間切換
- **現代化圖形界面**：使用 egui 建立響應式用戶界面

## 安裝說明

### 必要條件

- Rust 工具鏈（建議使用最新穩定版本）

### 建置項目

```bash
cargo build --release
```

### 執行應用程式

```bash
cargo run
```

## 使用方式

1. 在輸入欄位中輸入數學表達式
2. 點擊「計算」或按下 Enter 鍵
3. 在結果欄位中查看結果
4. 使用「歷史記錄」按鈕查看先前的計算
5. 切換「安全模式」以控制運算限制

## 數學函數

計算器支援廣泛的函數：

- 基本運算：`+`、`-`、`*`、`/`、`%`、`^`
- 常數：`pi`、`e`
- 三角函數：`sin`、`cos`、`tan` 等
- 對數函數：`log`、`log10`、`ln`

## 架構設計

應用程式分為幾個模組：

- `calculator`：核心計算邏輯
- `ui`：圖形用戶界面
- `config`：應用程式設定
